# Links

<Steps>
    ### Go to dashboard

    Visit [Portfolio4me.com Dashboard](https://portfolio4me.com/dashboard) and scroll down to the "Links" section.

    ### New link wizard

    Click on the "Add new link" button to open the new link wizard.

    ### Customize

    Select the type of link you want to add and provide the necessary information.

    ### Save

    Click the "Add" button to save your new link.

    ### Edit or delete

    After saving, you can edit or delete the link at any time. To edit, click on the edit icon. To delete, click on the delete icon.
</Steps>
